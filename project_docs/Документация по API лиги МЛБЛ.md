## Получение данных игрока
### Получение ID игрока
`https://{region_league}.ilovebasket.ru/players/{player_id}?apiUrl=https://reg.infobasket.su&compId={comp_id}&lang={lang}`
Из URL необходимо извлечь:
1. `player_id` - как path-параметр, id игрока
2. `comp_id` - как query-параметр, какой-то параметр необходимый для корректного получения данных
3. `region_league` - название региона или лиги
4. `lang` - код языка страницы
>[!info] Пример (GET)
>https://moscow.ilovebasket.ru/players/408436?apiUrl=https://reg.infobasket.su&compId=88649&lang=ru
